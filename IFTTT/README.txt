Some IFTTT examples
===================

- send_ifttt.sh		send an alarm with sensor-id and temperature
- temp_alarm_mcp9808.sh send an alarm using th MCP9808
- temp_alarm_bme688.sh	send an alarm using th BME68x
- eco2_alarm_bme688.sh	send an alarm using th BME68x (using eCO2 value)

To get you key, you must search "Webhooks" from ifttt.com then click on "Documentation" !
